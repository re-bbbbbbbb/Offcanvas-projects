Barra laterale - istruzioni

Per creare la barra laterale si usa il tag:

  • div class="offcanvas offcanvas-start show" id="demo" style="visibility: visible;" aria-modal="true" role="dialog"

Altri esempi di offcanvas sono:

  • .offcanvas => è la classe che crea la barra laterale offcanvas.
  • .offcanvas-start => è la classe che posiziona l'offcanvas e lo rende largo 400 px.
  • .offcanvas-title => è la classe che garantisce margini e altezza della linea corretti.
  • .offcanvas-body => è la classe con cui puoi aggiungere i tuoi contenuti all'interno della barra laterale.
  • I tag button e a => servono per aprire la barra laterale offcanvas, elementi presenti nel tag .offcanvascontenitore.

I tag che si usano per posizionare l'area offcanvas a sinistra, a destra, in alto o in basso sono:

  • .offcanvas-end => posiziona la barra a destra.
  • .offcanvas-start => posiziona la barra a sinistra.
  • .offcanvas-top => posiziona la barra in alto.
  • .offcanvas-bottom => posiziona la barra in basso.


L'ultimo bottone citato si crea usando il seguente tag:

  • button class="btn btn-primary" type="button" data-bs-toggle="offcanvas" data-bs-target="#demo"

Offcanvas barra laterale

Offcanvas viene spesso utilizzato come menu di navigazione della barra laterale.

Per creare il bottone bisogna usare il tag:

  • button class="btn btn-primary" type="button" data-bs-toggle="offcanvas" data-bs-target="#demo"
